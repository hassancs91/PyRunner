Using this data store in scripts

from pyrunner_datastore import DataStore store = DataStore("{{ datastore.name }}") # Set values store["my_key"] = {"any": "json value"} # Get values value = store["my_key"] value = store.get("my_key", default=None) # Check if key exists if "my_key" in store: print(store["my_key"]) # Delete a key del store["my_key"] # List all keys for key in store.keys(): print(key, store[key])

REST API access

Access this datastore via the REST API using an API token.

Authentication

Include your token in the request header:

Authorization: Bearer YOUR_API_TOKEN # or X-API-Key: YOUR_API_TOKEN

Get datastore info

curl -H "Authorization: Bearer YOUR_TOKEN" \ "{{ request.scheme }}://{{ request.get_host }}/api/v1/datastores/{{ datastore.name }}/"
Sample response { "name": "{{ datastore.name }}", "description": "{{ datastore.description|default:"" }}", "entry_count": {{ entries.count }}, "created_at": "{{ datastore.created_at|date:"c" }}", "updated_at": "{{ datastore.updated_at|date:"c" }}" }

List entries (paginated)

curl -H "Authorization: Bearer YOUR_TOKEN" \ "{{ request.scheme }}://{{ request.get_host }}/api/v1/datastores/{{ datastore.name }}/entries/?page=1&page_size=50"
Sample response { "entries": [ { "key": "example_key", "value": {"any": "json data"}, "created_at": "2026-01-20T08:00:00Z", "updated_at": "2026-02-01T12:00:00Z" } ], "count": {{ entries.count }}, "page": 1, "page_size": 50, "total_pages": 1 }

Get single entry

curl -H "Authorization: Bearer YOUR_TOKEN" \ "{{ request.scheme }}://{{ request.get_host }}/api/v1/datastores/{{ datastore.name }}/entries/YOUR_KEY/"
Sample response { "key": "example_key", "value": {"any": "json data"}, "created_at": "2026-01-20T08:00:00Z", "updated_at": "2026-02-01T12:00:00Z" }

List all datastores

curl -H "Authorization: Bearer YOUR_TOKEN" \ "{{ request.scheme }}://{{ request.get_host }}/api/v1/datastores/"
Sample response { "datastores": [ { "name": "{{ datastore.name }}", "description": "{{ datastore.description|default:"" }}", "entry_count": {{ entries.count }} } ], "count": 1 }
