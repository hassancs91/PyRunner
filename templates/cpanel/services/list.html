How to use in a script

Install claude-agent-sdk in the script's Environment, then:

from pyrunner_ai import ask_claude

answer = ask_claude("Search the web for today's AI news and summarize it")
print(answer)

For a subscription token, run claude setup-token on your own machine (logged into Claude). Paste the authorization code from the browser back into that terminal, then copy the token it prints (starts with sk-ant-oat01-) into the field above — not the browser code itself.
